# Accenture Stream Training – Azure Cloud Fundamentals

This repository contains my personal notes, summaries, and learning resources from the **Accenture Udacity Azure Cloud Fundamentals** course. These notes are created to help reinforce concepts learned during the training and serve as a quick revision guide.

## Course Overview

The Azure Cloud Fundamentals syllabus includes:

1. Cloud Computing
2. Foundational & Compute Services
3. Storage, Content Delivery, and Security
4. Networking, Elasticity, Messaging, and Containers
5. Getting Started – Azure Setup
6. Introduction to Microsoft Azure Development
7. Azure Compute Services
8. Azure Storage
9. Security and Monitoring Basics
10. Introduction to Azure Microservices
11. Serverless Functions
12. Enterprise Logic Apps and Event Grid
13. Deployment
14. Introduction to Security in Azure
15. Identity, Authentication, and Access (IAM)
16. Infrastructure and Network Security
17. Data Security and Protection
18. Protect and Monitor Cloud Resources
19. Designing Infrastructure and Managing Migration
20. Azure Cloud Capabilities
21. Designing for Backup and Recovery
22. Migration
23. Automation
